# OHS AI Portfolio Starter Kit

## Projects
1. **Workplace Injury Risk Prediction** — Predict incidents using worker and environment data.
2. **Air Quality Forecasting** — Forecast PM2.5 trends using Prophet.
3. **Heat Stress Anomaly Detection** — Identify physiological risk from IoT sensor data.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Open each notebook in Jupyter or VS Code.
3. Run all cells and view charts + metrics.

## Purpose
This starter kit demonstrates how AI, IoT, and data science enhance Occupational Health & Safety practices.
